我已经想"开始写点什么"很久了。

每次冒出来一个想法的时候，会觉得"这个值得记下来"，然后过几天就忘了。
有时候是在散步、洗碗、或者地铁上，那种半成形的句子很容易溜走。
我想要一个地方，能让这些东西不至于被生活的节奏冲掉。

## 为什么不是公众号

我考虑过公众号、即刻、小红书这些。它们的好处是有现成的读者，
坏处是——我每写一段东西，都要去那个平台才能找到它。
平台改了规则，内容就不再属于我；想把同一篇东西放到几个地方，
就要不停地复制粘贴；时间一长，我自己都不知道哪一份是"最新的"。

我想要一个**统一的入口**。一个我自己拥有的地方。
URL 是我的，内容是我的，格式是我的。
其它平台如果想要，就让链接指向这里。

## 为什么不是更大的目标

我没有想过靠这个赚钱，至少现在没有。
我也没有想过要写出什么体系化的东西。

我观察过自己——任何一件事，只要一开始就背着"应该做大"的预期，我就会动作变形。
所以这次我想换一种方式：
**先把"持续地写"这件事做成，再去想其它。**

这意味着：

- 不一定每篇都长。八百字、一千字，是我觉得舒服的长度。
- 不一定每篇都"有用"。有时候只是一个观察、一个情绪。
- 不一定每篇都对。我的看法会变，那就让它变。

## 关于这个博客本身

这个站是用 Astro 搭的，文章是 Markdown 文件，托管在 Vercel 上。
我和 Claude Code 一起写——我把脑子里零碎的想法说出来，
它帮我整理成结构化的文字，然后推到代码仓库，自动部署。

这套流程对我来说有点重要：
**摩擦越小，我越有可能持续。**
我希望"想到 → 落到字面上 → 发布"这条路尽量短，
不要被工具、平台、格式拖住。
